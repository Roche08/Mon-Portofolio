Application de Gestion de Tâches (Java)

Technologies utilisées : Java, Swing, Collections, Fichiers

Une application permettant de gérer des tâches, avec des options pour ajouter, supprimer et marquer des tâches comme terminées. Les tâches sont sauvegardées dans un fichier texte pour la persistance des données.


// Exemple d'ajout de tâche
public void addTask(String task) {
    tasks.add(task);
    saveToFile();
}

private void saveToFile() {
    try (BufferedWriter writer = new BufferedWriter(new FileWriter("tasks.txt"))) {
        for (String task : tasks) {
            writer.write(task);
            writer.newLine();
        }
    } catch (IOException e) {
        e.printStackTrace();
    }
}

Script d'Automatisation (Python)

Technologies utilisées : Python, Requests, BeautifulSoup, CSV

Un script pour extraire des données depuis un site web et les enregistrer dans un fichier CSV pour une analyse ultérieure.


import requests
from bs4 import BeautifulSoup
import csv

url = 'https://example.com'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# Extraction des titres
titles = soup.find_all('h2')

# Sauvegarde des titres dans un fichier CSV
with open('titles.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Title'])
    for title in titles:
        writer.writerow([title.get_text()])
